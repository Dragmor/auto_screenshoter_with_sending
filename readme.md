Программа предназначена для отправки скриншотов экрана ПК в реальном времени в ВКонтакте или Телеграм, или одновременно в оба сервиса. В интерфейсе программы вы можете указать интервал, с которым требуется делать скриншоты, а также предоставить токены для ВКонтакте или Телеграм. После нажатия кнопки "Старт", программа переходит в режим работы, не отображая свой графический интерфейс.

Преимущества программы:

*Удобство использования: Программа работает в фоновом режиме, не отвлекая пользователя своим графическим интерфейсом.
*Мгновенная доставка: Скриншоты мгновенно отправляются в выбранный сервис (ВКонтакте или Телеграм) и сохраняются в соответствующих диалогах.
*Автоматический запуск: При выборе опции "Добавить в автозагрузку", программа автоматически запускается в скрытом режиме при каждом включении компьютера.
*Простая настройка: После запуска программы можно легко изменить настройки, следуя нескольким простым шагам.

Инструкция по изменению настроек программы:
1)    Завершите процесс "Shoter.exe" в диспетчере задач.
2)    Откройте папку с программой.
3)    Дважды щелкните по исполняемому файлу программы (exe-файл).
4)    Откроется окно настроек программы.
5)    После внесения необходимых изменений нажмите кнопку "Старт".
6)    Готово!
